# Jogo do Número Secreto

Projeto desenvolvido durante os estudos de lógica de programação e Git/GitHub na Alura.

O objetivo do jogo é descobrir o número secreto gerado aleatoriamente pelo sistema. A cada tentativa, o jogador recebe dicas informando se o número secreto é maior ou menor que o chute informado.

## Funcionalidades

- Geração de número secreto aleatório
- Entrada de tentativas pelo usuário
- Dicas para orientar o jogador
- Contador de tentativas
- Reinício do jogo
- Intervalo de números de 1 a 100

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
